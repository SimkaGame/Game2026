# Portal Jumper

Portal Jumper — это 2D-платформер, в котором главный герой прыгает по платформам, сражается с монстрами и ищет порталы, чтобы попасть на следующий уровень.  
Каждый новый портал переносит персонажа в другой мир с новыми врагами, ловушками и испытаниями.  
Цель — пройти все уровни.

## О проекте

Учебный проект на C#

## Основные механики, которые планируется реализовать:
- управление персонажем (движение, прыжок, атака)
- враги с простым ИИ
- система уровней через порталы
- сбор предметов / очков
